# 블랑 재고관리 - 실사용 서버 버전

## 포함 기능
- 직원 로그인 (초기 계정 `admin / 1234`)
- 제품 추가, 수정, 삭제
- 매입 등록과 재고 자동 증가
- 출고 등록과 재고 자동 감소 및 재고 부족 방지
- 거래명세서/매입장 사진 서버 저장
- 재고 현황, 부족 알림, 매입/출고 내역 조회
- SQLite 데이터베이스로 기록 영구 저장

## 컴퓨터에서 실행하기
1. Node.js 20 이상 설치
2. 이 폴더에서 터미널 열기
3. `npm install`
4. `npm start`
5. 브라우저에서 `http://localhost:3000` 열기

## 휴대폰에서도 사용하려면
이 프로젝트를 Render, Railway, Fly.io, AWS Lightsail 등 서버에 배포하고 도메인을 연결해야 합니다.

## 배포 전 꼭 바꿀 것
- `server.js`의 세션 비밀키를 환경변수 `SESSION_SECRET`로 설정
- 초기 관리자 비밀번호를 변경
- HTTPS 도메인에서 운영할 때 `cookie.secure: true` 적용
- 사진과 DB를 영구 저장하는 디스크/스토리지를 선택
